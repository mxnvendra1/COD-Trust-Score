# 📦 The COD Trust Score

A data-driven storytelling dashboard built with Streamlit to manage Cash-on-Delivery (COD) returns in Indian e-commerce.

This project uses diagnostic and descriptive analytics, followed by predictive modeling (Gradient Boosting), to create a CIBIL-style trust score that assigns risk-based COD policies. 

Designed with a hyper-minimalist, editorial visual identity.

## 🌐 How to Deploy
1. Push these files to a GitHub repository.
2. Go to [share.streamlit.io](https://share.streamlit.io/).
3. Deploy pointing to `app.py`. The synthetic dataset will automatically generate on the first run.
